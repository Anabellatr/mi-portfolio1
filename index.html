Tercer proyecto

Como Analista de Sistemas use las siguentes tecnologías para el diseño y modelado del proyecto. UML (Unified Modeling Language): para crear diagramas que representen el sistema, incluyendo diagramas de casos de uso, de clases, de secuencia, y de actividades. BPMN (Business Process Model and Notation): para modelar los procesos de negocio, ideal para proyectos de reingeniería de procesos. Herramientas de modelado: Enterprise Architect: popular para la creación de diagramas UML. Lucidchart: para modelar procesos, sistemas y arquitecturas de manera colaborativa. Draw.io: una herramienta gratuita para crear diagramas UML, ER y BPMN. Visual Paradigm: para modelado de sistemas y procesos.
